# AI Interview System 🤖

## 📌 Overview

This project simulates a basic AI-powered interview system. It asks technical questions and evaluates user answers to provide a score and feedback.

## 🚀 Features

* Automated question asking
* Answer evaluation using keyword logic
* Score calculation
* Final performance feedback

## 🛠 Tech Stack

* Python

## ▶️ How to Run

1. Clone the repository
2. Run:
   python app.py
3. Answer the questions in terminal

## 📊 Output

* Score for each answer
* Final result (Excellent / Good / Needs Improvement)

## 💡 Future Improvements

* NLP-based evaluation
* Web interface using Flask
* Voice-based interview system
